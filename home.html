PennBook

Music-themed social media site

As the final project for my Networks and Cloud Computing class, I designed and coded a Facebook clone in node.js with Amazon AWS's dynamoDB serving as a database. Its features include profile pages, status updates, friend requests, a live chat, and a friend recommendation algorithm, as well as a network visualizer run through Hadoop MapReduce.

Echo

The best wake-up call is yourself

(Work in progress) Coded in React Native, Echo is an iOS lifestyle app that allows you to create less invasive alarms with an audio message.

After users record an audio clip that includes their desired time, the app uses voice-recognition software to set a corresponding ping at that time.

SearchLt

Chrome extension for convenient page searching

This extension provides users with an alternative to chrome's native page search. Based off of a feature of many popular programming IDEs, users can double-click or select a term to highlight references throughout the page, as well as show key search terms if navigating to a page from a search engine.

SideWinder

Multiplayer twist on the classic game of snake

SideWinder was my first exposure to working with React Native. Two players each control a side of a single screen in this take on a classic.

The objective of the game is to derail the other player's snake while collecting power-ups and avoiding collision with the walls, your tail, or the other snake.

DiskReader

DevOps application for simplifying I/O operations

DiskReader is a Windows application that allows engineers to easily read in the properties of a selected disk or system of disks.

I designed and created the application as an alternative solution to classifying the various types of disks that the Unisys software ran on. Previously, the only way to identify a disk was to inspect it through a Hex editor. Built with Visual Studio's WPF framework and a C# backend.

HexEd

redesigning a Hex editor for improved functionality

As an intern for Unisys, I added the ability to open physical drives (such as C:\) to their existing Hex editor tool.

After adding in an option for engineers to create handles to all physical drives on a system to view them, I made several improvements to the UI including line numbers, byte navigation, and optimized the scrollbar to handle large file sizes.

BreathaLocker

arduino-powered solution to drunk riding

A classmate and I drew got the inspiration for our Intro to Bioengineering final project after hearing news that a fellow student was injured biking home from an off-campus party

We created BreathaLocker, a bike lock powered by an Arduino Uno that uses a built-in breathalyzer to detect if the owner is able to ride, lifting a latch and exposing the keyhole accordingly.
